Welcome to xtool! 
==================================

## 安装方法
- 下载此仓库到本地
- ```cd xtool```
- ```pip install -e .```

## 使用方法
- ```from xtool.xp import xp```
- ```xp.feed(inputs)```
- ```xp.collect(locals()[, pre=''])```
- ```xp.p('tensor_var'[, num=20])```
